## 106. Construct Binary Tree from Inorder and Postorder Traversal

<h3><a href="https://leetcode.com/problems/construct-binary-tree-from-inorder-and-postorder-traversal/">leetcode-106</a></h3>

Input:
Inorder: [9, 3, 15, 20, 7]

Postorder: [9, 15, 7, 20, 3]
Output:

```plaintext
    3
   / \
  9  20
    /  \
   15   7

```
