## Convert a binary tree to a singly linked list.

Convert a binary tree linked to a linked list, after conversion, the left of the tree should be null.
Use pre-order traversal.

For example:
Input:

```bash
        1
       / \
      2   5
     / \   \
    3   4   6

```

Output:

```bash
1
 \
  2
   \
    3
     \
      4
       \
        5
         \
          6

```
